TAG dl
A tag dl serve para criar uma "description list"(lista de descrição). Ela usa as tags dt e dd para exibir dois termos relacionados.
TAG ul
A tag ul é um elemento HTML que representa uma lista não ordenada.
TAG ol
O Elemento HTML ol (ou Elemento HTML de lista ordenada) representa uma lista de itens ordenados.
